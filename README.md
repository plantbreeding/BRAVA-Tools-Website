# BRAVA Tools Website

Website to use with the [BRAVA Tools](https://github.com/plantbreeding/BRAVA-Tools) backend.


## Tech Stack
* [Vue](https://vuejs.org/)
* [Vite](https://vite.dev/)
* [PrimeVue](https://primevue.org/) (Component Library)
* [Tailwind CSS](https://tailwindcss.com/)

## Prerequisites

* npm 11.6.1
* node 22.20.0

## Run/Build

```bash
npm install
```

For development, start a server:

```bash
npm run dev
```

To build for a use in a Dockerfile:

```bash
npm run build
```
